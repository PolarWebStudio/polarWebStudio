1. Servicios Exprés y Catálogos

  • El tiempo de entrega estimado es de 2 a 4 horas una vez recibida toda la información.
  • Estos planes no poseen cambios ilimitados, incluye hasta 2 cambios mensuales. Otos cambios adicionales tienen costo extra.

  • Los servicios exprés no incluyen funcionalidades avanzadas ni integraciones personalizadas.
  • No se ofrecen servicios ilimitados ni soporte técnico continuo para estos planes.

2. Proyectos Web Corporativos y E-commerce

  • Se requiere un 50% de anticipo para iniciar el proyecto.
  • El 50% restante se paga al momento de la entrega final.
  • Se incluyen hasta 2 revisiones durante el desarrollo.
  • El cliente debe entregar el material en un máximo de 5 días hábiles.

3. Condiciones Generales

  • El servicio incluye únicamente desarrollo web, no gestión de ventas ni atención al cliente.
  • El cliente es responsable del contenido (textos, imágenes, logos).
  • El dominio (.com / .co) tiene un costo adicional anual.
  • El código es propiedad de Polar Web Studio hasta completar el pago total.
